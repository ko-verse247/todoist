# Todoist

## Overview

Simple Todo list web app.

## Frontend (client)

[Deployed on Render.](https://todoist-task.onrender.com/)

### Tech used

- Vue.js
- PUG
- Less
- Day.js
- js-confetti
- Axios


## Backend (server)

[Deployed on Render.](https://todoist-be.onrender.com/)

### Tech used
- Node.js
- Express.js
- MongoDB
- Mongoose
- Nodemon
- dotenv
- Prettier
- ESLint
